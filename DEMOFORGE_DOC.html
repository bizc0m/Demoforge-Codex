Niveaux de complexité

Chaque effet doit adapter son panneau selon un niveau numérique et verbal.

0 — Noob Très simple, presets, texte et bouton export.
3 — Sceener Réglages utiles, paramètres visibles, rythme de scène.
7 — Leet Contrôle fin, valeurs numériques, comportement avancé.
10 — R0x0r Paramètres internes, raw, contraintes hardware et export total.

Interface d'idée : texte, fontes et raster

Quand c'est pertinent, chaque effet peut exposer du texte, un sélecteur de fonte, et des couleurs type raster / palette.

Effets : une section dédiée par effet

Chaque effet doit disposer de sa propre section UI, de son bloc d'attribution, de ses paramètres, et de son niveau recommandé.

Scroller / Texte

Texte éditable, fonte choisissable, vitesse, taille, palette, modes noob → r0x0r.

  • texte changeable
  • sélecteur de fontes
  • couleurs / raster / gradient
  • export versionné

Raster Bars / Copper Bars

Palette changeable, hauteur des barres, vitesse, synchro musique, attribution historique.

Plasma

Palette, résolution, vitesse, formule simple ou avancée selon le niveau.

Starfield

Densité, vitesse, profondeur, couleurs, mode simple ou expert.

Fire / ASCII / NFO

ASCII et textmode intégrables quand le style le demande, avec fontes dédiées.

Preset intégrable

On doit pouvoir intégrer un preset externe depuis un repo ou un fichier standardisé.

  • repo Git donné par l'utilisateur
  • preset JSON versionné
  • format documenté pour rendre un preset intégrable
  • attribution incluse

Format suggéré

{
  "preset_id": "cracktro_amiga_1991",
  "title": "Cracktro Amiga 1991",
  "level": 3,
  "effects": [
    { "id": "copper_bars" },
    { "id": "dycp_scroller" }
  ],
  "fonts": ["amiga_topaz_1200"],
  "music": {
    "type": "external",
    "url": "https://..."
  }
}

Exports prévus

Export HTML Pure

Sortie HTML autonome, lisible et modifiable.

Export HTML Packer

Version compressée / packée pour diffusion ou challenge taille.

README export source

Documentation exportée avec sources, crédits et instructions.

Modes d'export de production

  • Export Demo : version simple, orientée lecture / diffusion
  • Export Demo Pedago : version avec explications et attribution
  • Export Demo Pedago + Source : version éducative et modifiable en temps réel

Tous les exports doivent rester exportables ensuite et idéalement réimportables.

Musique

  • taper une URL de musique extérieure
  • intégrer la musique directement dans le fichier exporté
  • garder une source lisible
  • supporter un mode simple et un mode scène / tracker

Types de musique

{
  "music": {
    "mode": "external_or_embedded",
    "url": "https://example.com/demo.mod",
    "embed": true,
    "source_label": "ModArchive",
    "source_url": "https://modarchive.org/"
  }
}

  • La musique peut être externe ou intégrée
